# TESTES VERSIONAMENTO NO GIT E GITHUB

### Criação de um arquivo (PYTHON) - "calculos.py"

> Comandos Utilizados:
* git init (Criação do diretório)
* git add <arquivo> (Adicionando o arquivo a ser versionado)
* git commit -m "Mensagem" (Comitando/criando uma comit um versão do arquivo)
* git branch -U <nome_da_branch> (Criando uma nova Branch / Raiz)
* git status (situação do arquivo na branch atual)
* git log (verificação dos commits criados)
* git remote add <branch> https://... (Preparando o arquivo para ser enviado a branch remota)
* git push (envio da commit local para branch remota)
* git pull (recebendo commit remota na branch local)
